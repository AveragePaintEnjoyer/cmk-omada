# Setup in MSP Mode

Login into your msp controller and go to `Settings > Platform integration`.

![](./../.assets/integration-msp.png)

Next add new app, give it a name and view permissions.

![](./../.assets/msp-1.png)

Copy details to config part of script:

- CLIENT ID
- CLIENT SECRET
- Interface Access Address
- OMADA ID
